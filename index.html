Vectorious 2

Vectorious is a generalized n-dimensional matrix and vector library written in JavaScript, which can be used both in node.js and the browser.

Installation

In Node.js:

Install with npm

$ npm install vectorious

Test with

$ mocha

In the browser:

Download dist/vectorious-2.0.0.js and use it like this:

<script src="vectorious-2.0.0.js"></script>
<script>
  // e.g. var vector = new vectorious.Vector()
</script>

Extensions

  • vectorious-solve - Solves linear systems of equations of the form Ax = B.
  • vectorious-plot - Generates a two-dimensional SVG plot from two input vectors.

Usage

The constructors of both Matrix and Vector are designed to be flexible, so they can be initialized using several different arguments.

var vectorious = require('vectorious');

var vector,
    matrix;

vector = new vectorious.Vector();
// { values: [], length: 0 }

matrix = new vectorious.Matrix();
// { rows: [] }

vector = vectorious.Vector.zeros(5);
// { values: [0, 0, 0, 0, 0], length: 5 }

vector = new vectorious.Vector(1, 2, 3, 4, 5);
// { values: [1, 2, 3, 4, 5], length: 5 }

matrix = new vectorious.Matrix(vector);
// { rows: [ { values: [1, 2, 3, 4, 5], length: 5 } ] }

matrix = vectorious.Matrix.zeros(2, 2);
/* {
  rows: [
    { values: [0, 0], length: 2 },
    { values: [0, 0], length: 2 }
  ]
} */

var input = [
  [1, 2],
  [3, 4]
];

matrix = new vectorious.Matrix(input);
/* {
  rows: [
    { values: [1, 2], length: 2 },
    { values: [3, 4], length: 2 }
  ]
} */

  • add
  • subtract
  • scale
  • normalize
  • dot
  • magnitude
  • angle
  • project
  • zeros
  • ones
  • range
  • equals
  • get
  • min
  • max
  • set
  • combine
  • push
  • map
  • each
  • toString
  • toArray
